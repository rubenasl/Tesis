Inflexiones de 'software' (nm): mpl: softwares

Diccionario de la lengua española © 2005 Espasa-Calpe:

software

  1. (voz i.) m. inform. Término genérico que se aplica a los componentes no físicos de un sistema informático, como p. ej. los programas, sistemas operativos, etc., que permiten a este ejecutar sus tareas:
    compré el ordenador con el software necesario para llevar la contabilidad del negocio.
